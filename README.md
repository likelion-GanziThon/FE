# 커밋 컨벤션

| 타입         | 설명                                              |
| :----------- | :------------------------------------------------ |
| **feat**     | 새로운 기능 추가                                  |
| **fix**      | 버그 수정                                         |
| **refactor** | 리팩토링(동작 변화 없음)                          |
| **docs**     | 문서만 변경                                       |
| **test**     | 테스트 추가/변경                                  |
| **chore**    | 기타 잡일(코드 영향 거의 없음)                    |
| **rename**   | 파일/폴더명 변경/이동만 수행                      |
| **remove**   | 파일 삭제만 수행                                  |
| **build**    | 빌드 시스템/도구 변경 (e.g., vite, metro, gradle) |
